# 🚀 TaskFlow

**TaskFlow** es un sistema en desarrollo para la empresa **NovaTech**. Este proyecto se inicializa como una aplicación de consola y servirá como base para la gestión eficiente de flujos de trabajo y tareas.

---

## 📖 Descripción del Proyecto

TaskFlow busca solucionar las necesidades operativas de NovaTech a través de un sistema robusto y escalable. En esta primera etapa, el repositorio contiene la estructura base del proyecto configurada como una aplicación de consola, junto con su entorno de trabajo colaborativo para el equipo de desarrollo.

## 🛠️ Tecnologías Usadas

- **Lenguaje:** C#
- **Framework:** .NET (Aplicación de Consola)
- **Control de Versiones:** Git & GitHub
- **IDE Recomendado:** Visual Studio 

## ⚙️ Cómo ejecutar el proyecto

Para correr esta aplicación en tu entorno local, sigue estos pasos:

1. **Clonar el repositorio:**
   `git clone https://github.com/jordansep/taskflow`

2. **Navegar a la carpeta del proyecto:**
   `cd taskflow`

3. **Abrir el proyecto:**
   Abre la solución (.sln) en Visual Studio.

4. **Restaurar dependencias y compilar:**
   Si usas la terminal de .NET, ejecuta: `dotnet build`

5. **Ejecutar la aplicación:**
   Si usas la terminal de .NET, ejecuta: `dotnet run`
